# Project Alpha Frontend

React + TypeScript + Vite + Tailwind CSS + Shadcn UI 前端应用。

## 安装

```bash
npm install
```

## 开发

```bash
npm run dev
```

应用将在 http://localhost:5173 启动。

## 构建

```bash
npm run build
```

## 预览生产构建

```bash
npm run preview
```

## 技术栈

- **React 18**: UI 库
- **TypeScript**: 类型安全
- **Vite**: 构建工具
- **Tailwind CSS**: 样式框架
- **Shadcn UI**: 组件库
- **Zustand**: 状态管理
- **Axios**: HTTP 客户端
